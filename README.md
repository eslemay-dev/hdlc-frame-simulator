# HDLC Frame Simulator

This project simulates an HDLC-like data link layer protocol.

## Features

- CRC calculation and validation
- Bit stuffing and destuffing
- Frame generation with flags
- Error detection simulation
- GUI-based interface

## Technologies

- Python
- Tkinter

## How to Run

```bash
python main.py
```

## Description

This project demonstrates how data frames are created, transmitted, and validated in data communication systems using HDLC-like structures.

## Concepts Covered

- Data Link Layer
- Error Detection (CRC)
- Bit Stuffing
- Frame Synchronization
